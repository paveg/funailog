## アクセス解析ツールについて

当サイトでは、サイトの利用状況を把握するために以下のアクセス解析ツールを使用しています。

### Cloudflare Web Analytics

当サイトはCloudflare Web Analyticsを使用しています。このツールはプライバシーに配慮した設計となっており、Cookieを使用せず、個人を特定する情報を収集しません。収集されるデータはページビュー数、訪問者数、リファラー、国・地域などの集計情報のみです。

詳細については[Cloudflareのプライバシーポリシー](https://www.cloudflare.com/privacypolicy/)をご覧ください。

### Google Search Console

当サイトはGoogle Search Consoleを使用して、検索エンジンにおけるサイトのパフォーマンスを分析しています。これにより、検索クエリ、クリック数、表示回数などのデータを収集していますが、個人を特定する情報は含まれません。

詳細については[Googleのプライバシーポリシー](https://policies.google.com/privacy?hl=ja)をご覧ください。
