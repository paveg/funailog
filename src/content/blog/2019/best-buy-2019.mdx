この記事は「今年買ってよかったもの Advent Calendar 2019」の9日目です。

こんにちは、funaiです。2019年も残りわずかとなりました。Amazonの注文履歴を振り返ってみると、今年は**106件・約230,270円**の買い物をしていました。

この記事では、その中から本当に買ってよかったと思える**ベストバイ5選**を紹介します。

## 5位: Nature Remo レビュー｜スマートホーム化の第一歩

**購入時期**: 3月 / **使用期間**: 9ヶ月 / **価格**: 約¥12,000

スマートホーム化の第一歩として購入したのが**Nature Remo**です。

https://nature.global/

### おすすめポイント

- **外出先からエアコンを操作** - 帰宅前に部屋を暖めておける
- **スケジュール設定** - 毎朝決まった時間にエアコンON
- **Google Home / Alexa連携** - 声で家電を操作

### デメリット・注意点

- 赤外線が届く範囲のみ対応
- 初期設定にやや手間がかかる

特に冬場は、帰宅前にエアコンを付けておけるのが本当に便利です。「寒い部屋に帰る」というストレスがなくなりました。

## 4位: タワー型電源タップ｜スマートホームの必需品

**購入時期**: 4月 / **使用期間**: 8ヶ月 / **価格**: 約¥3,500

スマートホーム化を進めると、**コンセントが圧倒的に足りなくなります。**

### おすすめポイント

- 縦置きで**省スペース**
- **USB-Aポート搭載**で充電器不要
- 個別スイッチ付きで**待機電力カット**

現在**約15個のデバイス**を接続していますが、問題なく使えています。

## 3位: Sesame mini レビュー｜鍵を持ち歩かない生活へ

**購入時期**: 2月 / **使用期間**: 10ヶ月 / **価格**: 約¥15,000

**鍵を持ち歩かない生活**を実現してくれたのがSesame miniです。

https://jp.candyhouse.co/

### おすすめポイント

- **スマホで解錠・施錠** - 物理キー不要
- **オートロック機能** - 施錠忘れゼロ
- **施錠履歴の確認** - いつ誰が開けたかわかる

### デメリット・注意点

- 取り付けできない鍵もある（事前確認必須）
- WiFiモジュールは別売り
- 電池交換が必要（約1年）

「鍵を閉めたかな？」という不安がなくなりました。履歴で確認できるので、外出先でも安心です。

## 2位: Panasonic Cuble ドラム式洗濯乾燥機｜生活を変えた家電

**購入時期**: 1月 / **使用期間**: 11ヶ月 / **価格**: 約¥200,000

**2019年で最も生活を変えた家電**がドラム式洗濯乾燥機です。

### 導入前後の変化

| 項目         | 導入前             | 導入後             |
| :----------- | :----------------- | :----------------- |
| 洗濯物を干す | 必要               | **不要**           |
| 花粉シーズン | 部屋干しで生乾き   | **関係なし**       |
| 洗濯時間     | 干す作業込みで30分 | **ボタン押すだけ** |
| 仕上がり     | 天候に左右         | **常にふわふわ**   |

**洗濯物を干すという行為が人生から消えます**。これだけで導入する価値があります。

### おすすめポイント

- **完全自動化** - 洗濯〜乾燥まで放置でOK
- **花粉対策** - 外干し不要で花粉症の人に最適
- **時短効果** - 家事時間を大幅削減

### デメリット・注意点

- 設置スペースの確保が大変（賃貸は要確認）
- 初期費用が高い（20万円前後）
- 電気代はやや増加

それでも、時間と手間を考えれば**コスパは最高**だと断言できます。

## 1位: Triumph Street Triple RS｜2019年のベストバイ

**購入時期**: 5月 / **使用期間**: 7ヶ月 / **価格**: 約¥1,300,000

2019年のベストバイは**バイク**です。

### スペック

| 項目     | スペック            |
| :------- | :------------------ |
| 排気量   | 765cc               |
| 最高出力 | 123PS               |
| 車両重量 | 166kg               |
| 購入形態 | 試乗車（970km走行） |

ずっと欲しかったミドルクラスのスポーツバイク。試乗車として970km走行済みの個体を購入しました。

### なぜ1位なのか

**「欲しいものを買う」という行為そのものが幸せだった**からです。

ガジェットや家電は「便利になる」「効率が上がる」という合理的な理由がありますが、バイクは純粋に「**乗りたいから買った**」というもの。

風を感じながら走る体験は、お金では測れない価値があります。
