# U4025QW Review SEO Improvement: Title, Description, and Content Additions

## Overview

`src/content/blog/2024/review-monitors-u4025qw-dell.mdx` の SEO 改善。Search Console データで「5k2k モニター」「dell 5k2k」などの高インプレッション・低クリックなクエリを獲得していない問題に対し、frontmatter（title / description / lastUpdated）と本文の整合性確保、informational intent 対応のための解説セクション追加を行う。

## Motivation

Search Console の分析で以下の状況が確認されている：

1. **未獲得キーワード**: 「5k2k モニター」「dell 5k2k」で表示回数は出ているがクリック数 0。検索意図と現タイトルのマッチが弱い。
2. **PC ユーザー CTR の低さ**: PC（CTR 6.62%）はモバイル（9.65%）・タブレット（12.02%）より低い。エンジニア・ガジェット好きなど PC 上での検索者に対するタイトル/description の魅力不足。
3. **informational intent の欠落**: 「5k2k とは」のような解説クエリで露出があるが、現記事は製品レビューに特化しており解説要素がない。

この記事は funailog の gadgets カテゴリで主力レビュー記事の 1 つであり、CTR 改善の効果検証サイクルを早く回せる対象として最初に着手する。

## Non-Goals

- URL（slug）の変更: 既存被リンクへの影響回避のため `review-monitors-u4025qw-dell` のまま維持。
- アンカーリンク戦略全体の刷新: SEO 改善案 5（目次最適化・素材別記事への分割）は別 PR で扱う。
- 他のガジェットレビュー記事への横展開: パターン化は本 PR の効果検証後、別途実施。
- 価格・購入リンク情報の更新: AffiliateCard の現行値はそのまま維持。
- 新規記事の執筆: 「json-ldとは」など新規執筆は別セッションで扱う。
- コンテンツマーケティングスキル（content-marketing:content-marketer）の本格適用: 本 PR は既存記事への限定的改修であり、フル活用は別セッションで検討。

## Design Principles

`.claude/rules/blog-writing-style.md` および `~/.claude/rules/japanese-writing.md` に沿う：

1. **既存の文体と構成を尊重**: 個人レビューのトーン、丁寧語（です・ます調）、括弧内補足の余地を維持する。
2. **SEO 改修と文体保持のバランス**: 「【完全版】」「徹底解説」のようなクリックベイト系表現は採用しない。
3. **タイトルと本文の整合性**: タイトルが「2 年使った」と謳う以上、本文に長期使用の記述がなければ SERP との整合性が取れない（タイトル詐欺の回避）。

```mdx
## 5K2K とは？

5K2K（5120 × 2160 ピクセル）は、4K UHD（3840 × 2160）の左右に約 1280 ピクセルずつ広げた解像度です。アスペクト比は 21:9 のウルトラワイドに分類されます。

| 解像度 | ピクセル数  | アスペクト比 | 想定用途                           |
| :----- | :---------- | :----------- | :--------------------------------- |
| FHD    | 1920 × 1080 | 16:9         | 一般的な作業                       |
| 4K UHD | 3840 × 2160 | 16:9         | 映像視聴・写真編集                 |
| 5K2K   | 5120 × 2160 | 21:9         | マルチタスク・動画編集タイムライン |

4K と比べて横方向の作業領域が広がるため、デュアルモニターを 1 枚にまとめたい用途に向いています。一方で 16:9 のフルスクリーン動画では左右に黒帯が出ます。
```

**変更意図**:

- 「## 5K2K とは？」を独立 H2 として新設し、informational 検索クエリ（「5k2k とは」「5k2k 4k 違い」）への対応を強化。既存スペック表の前に配置し、「解像度の概念 → 製品の具体仕様」という論理的な流れにする。
- 「## What's U4025QW?」を「## DELL U4025QW のスペック」にリネーム。英語見出しを日本語化することで意味が明確化され、SEO 上もマイナスにはならない（外部からの被リンクリスクは低いと判断）。
- 比較表は 3 解像度のみに絞る（FHD / 4K UHD / 5K2K）。WQHD など中間解像度は冗長になるため省略。

```mdx
リフレッシュレート等をDell Display Managerから変更するとこの問題は回復します。いずれにしても、問題であることは変わりないので進展があれば追記することとします。
```

**After**: 上記に加えて以下のサブセクションを追加：

```mdx
### 2026/05 追記: 2 年使用後の感想

購入から 2 年以上が経過しました。以下、長期使用の所感です。

[ユーザーが実体験を埋める箇所]
```

**変更意図**: タイトル「2 年使った」の根拠を本文に持たせる。実体験ベースの内容はユーザー本人しか書けないため、spec ではテンプレート（埋めるべき項目）を定義し、実装時にユーザーが内容を記入する。
